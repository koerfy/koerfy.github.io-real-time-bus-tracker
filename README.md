# Real-time-bus-tracker
Description: This exercise allows for the manipulation of the DOM using asynchronous javascript functions and live data from the Massachusetts Bay Transportation Authority to track bus locations in Boston.

How to run: Fork the folder with the files. Download them and experiment on your own.

Future improvements: No future improvements.
